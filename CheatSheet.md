# CheetSheet 

## Python

### Virtual Environment

- create new virtual environment
```bash
python3 -m venv .venv
```

- activate virtual environment
    - Windows
    ```bash
    source .venv\Scripts\activate
    ```
    - MacOS/Linux
    ```bash
    source .venv/bin/activate
    ```

- deactivate virtual environment
```bash
deactivate
```
